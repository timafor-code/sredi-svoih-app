# Prototype files

Здесь лежат HTML/JavaScript-прототипы приложения «Среди Своих».

## sredi-svoih.html

Главный HTML/React-прототип приложения.

Используется как основной источник правды для:
- структуры экранов;
- нижней навигации;
- главной страницы;
- страницы молитв и зманим;
- страницы событий;
- страницы контактов;
- страницы профиля;
- вложенных экранов профиля;
- карточки контакта;
- модального окна Омера;
- визуального dark glass стиля.

При переносе в React Native запрещено использовать WebView или HTML.

Нужно реализовать нативные React Native компоненты.
