# Среди Своих — iOS-приложение

## Суть
Мобильное приложение для русскоязычной еврейской общины: еврейский календарь, времена молитв, зманим, дни рождения контактов с автоматической конвертацией в еврейский календарь, виджеты на главный экран iPhone.

## Важное уточнение для разработки
Описание ниже фиксирует продуктовую и техническую архитектуру. Если список экранов в этом документе расходится с HTML/JavaScript-прототипом, приоритет по интерфейсу и пользовательским сценариям имеет файл `docs/prototype/sredi-svoih.html`.

## Статус
Прототип собран в HTML/React. Разработка на React Native ещё не начата.

## Продуктовые решения
- Нусах выбирается пользователем в настройках.
- Язык интерфейса на старте — русский.
- Зманим считаются от выбранного города, а не автоматически от GPS, если пользователь явно выбрал город.
- GPS используется как опциональный источник предложения города.
- Мероприятия внутри общины: список событий, запись, количество мест, в будущем оплата регистрации.
- Контакты iPhone используются локально для дней рождения и расчёта еврейских дат.
- Серверная синхронизация контактов допустима только после явного согласия пользователя.

## Hebcal
Нужно подготовить слой `src/lib/hebcal.ts` и `src/lib/zmanim.ts` для:
- еврейской даты;
- недельной главы;
- праздников;
- счёта Омера;
- зманим;
- времени свечей;
- еврейских дней рождения;
- учёта рождения после заката.

## Supabase/PostgreSQL
Нужно подготовить Supabase-клиент, миграции и RLS. Минимальные домены:
- communities
- profiles
- community_contacts
- events
- event_registrations
- user_settings
- synced_contacts
- calendar_cache

## Privacy constraints
- Не отправлять все iPhone-контакты на сервер автоматически.
- Локальные контакты использовать локально, пока пользователь не дал consent.
- Для серверной синхронизации хранить минимум данных.
- Для phone/email в synced_contacts предпочтительно использовать hash.
- RLS обязателен для пользовательских данных.

## Риски
- Права на тексты сидура требуют отдельной проверки.
- Транслитерация Бейт Сфаради на кириллицу — отдельная фаза.
- Для иврита с никуд нужен подходящий шрифт.
- iPhone хранит дату рождения без времени; если рождение было после заката, нужна настройка `birth_after_sunset`.
