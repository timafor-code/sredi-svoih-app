# Среди Своих

iOS-приложение для русскоязычной еврейской общины.

## Стек

- Expo
- React Native
- TypeScript
- expo-router
- Zustand
- Supabase
- @hebcal/core
- expo-contacts
- expo-notifications
- expo-location

## Задача

Перенести HTML/React-прототип из `docs/Среди-Своих.html` в полноценное Expo React Native приложение.

Прототип используется как визуальная и продуктовая спецификация.  
WebView запрещён.  
HTML/CSS переносить напрямую запрещено.  
Нужна нативная iOS-реализация с тёмным glass-дизайном.

## Запуск

```bash
npm install
npx expo start
